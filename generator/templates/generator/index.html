O que é um arquivo AGENTS.md?

Um arquivo AGENTS.md instrui agentes de IA sobre como trabalhar com seu projeto. Geralmente é colocado na raiz do repositório e fornece instruções, convenções e contexto para ferramentas de IA.

Usos comuns:

  • Definir como executar, testar e fazer lint do projeto
  • Estabelecer convenções de código e estilo para agentes de IA
  • Documentar a estrutura e arquitetura do projeto
  • Fornecer contexto sobre dependências e frameworks
  • Especificar etapas de configuração do ambiente

Seções básicas:

  • run — Como iniciar o servidor de desenvolvimento
  • test — Como executar os testes
  • lint — Como verificar a qualidade do código
  • build — Como compilar para produção
  • style — Preferências de estilo de código
