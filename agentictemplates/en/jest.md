# Jest

## Commands

- Test: `npx jest` or `npm run test`
- Watch mode: `npx jest --watch`
- Coverage: `npx jest --coverage`

## Best Practices

- Use `describe`/`it` blocks for test organization
- Prefer `toBe` and `toEqual` matchers
- Use `jest.fn()` for simple mocks
- Use `jest.spyOn()` for partial mocks
- Write tests alongside implementation files
