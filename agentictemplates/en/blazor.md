# Blazor

## Commands

- Run: `dotnet run`
- Test: `dotnet test`
- Lint: `dotnet format`
- Build: `dotnet build`

## Code Style

- Use Blazor components with `.razor` files
- Prefer CSS isolation for component styles
- Use dependency injection in Blazor components

## Best Practices

- Choose render mode (Server vs WebAssembly) wisely
- Use `@rendermode` for interactive server rendering
- Implement state management with Fluxor or custom services
- Use virtualization for large lists
- Handle form validation with DataAnnotations
