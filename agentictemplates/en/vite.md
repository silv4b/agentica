# Vite

## Commands

- Run: `npm run dev` or `npx vite`
- Build: `npm run build` or `npx vite build`
- Preview: `npm run preview`

## Best Practices

- Use Vite's built-in asset handling and CSS processing
- Leverage HMR for fast development feedback
- Use environment variables with `import.meta.env`
- Configure build targets for browser compatibility
- Use plugins from the official Vite ecosystem
