# Nuxt.js

## Commands

- Run: `npm run dev`
- Build: `npm run build`
- Generate: `npm run generate`

## Code Style

- Use Nuxt 3 with auto-imports and composables
- Prefer `definePageMeta` for page metadata
- Use server routes for API endpoints
- Leverage Nitro for server engine

## Best Practices

- Use Nuxt layers for code sharing
- Use `useFetch` and `useAsyncData` for data loading
- Implement SEO with `useHead` composable
- Use middleware for route protection
- Leverage automatic code splitting
