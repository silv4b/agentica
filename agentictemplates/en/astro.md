# Astro

## Commands

- Run: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Code Style

- Use `.astro` components for static content
- Use islands architecture for interactive components
- Prefer content collections for blog/docs

## Best Practices

- Minimize client-side JavaScript with islands
- Use image optimization built-in
- Leverage view transitions for page navigation
- Use RSS and sitemap generation for content sites
