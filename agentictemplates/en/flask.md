# Flask

## Commands

- Run: `flask run` or `python app.py`
- Test: `pytest`
- Lint: `ruff check .`

## Code Style

- Follow PEP 8
- Use blueprints for modular organization
- Prefer factory pattern for app creation
- Use type hints for all public functions

## Best Practices

- Use Flask-SQLAlchemy for database access
- Use Flask-Migrate for schema migrations
- Implement proper error handlers and abort helpers
- Use Flask-Login for authentication
- Structure with application factory pattern
