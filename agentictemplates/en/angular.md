# Angular

## Commands

- Run: `ng serve`
- Test: `ng test`
- Lint: `ng lint`
- Build: `ng build`

## Code Style

- Follow the official Angular style guide
- Use standalone components (NgModules are legacy)
- Use signals for state management
- Prefer reactive forms over template-driven forms
- Use typed reactive forms

## Best Practices

- Use lazy loading for feature modules/routes
- Implement OnPush change detection strategy
- Use `async` pipe instead of manual subscriptions
- Follow the folder-by-feature structure
- Use Angular CLI for code generation
