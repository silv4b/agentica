# DaisyUI

## Commands

- Dev: `npm run dev`
- Build: `npm run build`

## Best Practices

- Use DaisyUI component classes (`btn`, `card`, `input`, `navbar`, etc.)
- Customize theme via `tailwind.config.js` with `daisyui.themes`
- Use semantic HTML with DaisyUI components
- Use theme names (`data-theme="dark"`, `data-theme="light"`) for theming
- Combine DaisyUI with Tailwind utility classes for fine-tuning
- Use `daisyui.config` for component-specific defaults
