# jQuery

## Melhores Práticas

- Use delegação de eventos com `.on()` para elementos dinâmicos
- Armazene seletores jQuery em cache em variáveis
- Encadeie métodos para legibilidade
- Prefira `.on()` em vez dos métodos obsoletos `.click()`, `.bind()`, etc.
- Use `$(document).ready()` para prontidão do DOM
- Considere JavaScript vanilla para suporte moderno de navegadores
