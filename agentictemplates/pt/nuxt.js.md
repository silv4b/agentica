# Nuxt.js

## Comandos

- Executar: `npm run dev`
- Build: `npm run build`
- Gerar: `npm run generate`

## Estilo de Código

- Use Nuxt 3 com auto-imports e composables
- Prefira `definePageMeta` para metadados de página
- Use server routes para endpoints de API
- Aproveite Nitro como motor de servidor

## Melhores Práticas

- Use camadas Nuxt para compartilhamento de código
- Use `useFetch` e `useAsyncData` para carregamento de dados
- Implemente SEO com composable `useHead`
- Use middleware para proteção de rotas
- Aproveite code splitting automático
