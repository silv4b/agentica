# NestJS

## Comandos

- Executar: `npm run start:dev`
- Testar: `npm run test`
- Linter: `npm run lint`
- Build: `npm run build`

## Estilo de Código

- Siga o guia de estilo oficial NestJS
- Use módulos para organização de funcionalidades
- Prefira classes com decorators em vez de objetos simples
- Use DTOs e pipes de validação

## Melhores Práticas

- Use injeção de dependência como padrão principal
- Organize código em módulos (baseado em funcionalidades)
- Use guards para autenticação/autorização
- Use interceptors para logging e transformação
- Use pipes para validação de entrada
- Use TypeORM ou Prisma para acesso a banco de dados
