# Astro

## Comandos

- Executar: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Estilo de Código

- Use componentes `.astro` para conteúdo estático
- Use arquitetura de ilhas para componentes interativos
- Prefira content collections para blogs/docs

## Melhores Práticas

- Minimize JavaScript do lado do cliente com ilhas
- Use otimização de imagem integrada
- Aproveite view transitions para navegação entre páginas
- Use RSS e geração de sitemap para sites de conteúdo
