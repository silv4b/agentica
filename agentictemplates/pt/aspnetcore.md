# ASP.NET Core

## Comandos

- Executar: `dotnet run`
- Testar: `dotnet test`
- Linter: `dotnet format`
- Build: `dotnet build`

## Estilo de Código

- Siga as convenções de código .NET
- Use namespaces em escopo de arquivo
- Use construtores primários quando apropriado
- Prefira records para objetos de transferência de dados

## Melhores Práticas

- Use injeção de dependência em toda a aplicação
- Implemente o padrão repository para acesso a dados
- Use middleware para preocupações transversais
- Configure logging estruturado (Serilog)
- Use Entity Framework Core para acesso a banco de dados
