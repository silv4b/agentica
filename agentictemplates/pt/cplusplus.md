# C++

## Comandos

- Build: `cmake --build .` ou `make`
- Testar: `ctest`

## Estilo de Código

- Siga as C++ Core Guidelines
- Use RAII para gerenciamento de recursos
- Prefira smart pointers em vez de raw pointers
- Use `constexpr` quando aplicável
- Use recursos C++20/23 quando possível

## Melhores Práticas

- Use CMake para build
- Separe arquivos de cabeçalho da implementação
- Use namespaces para evitar conflitos de nomes
- Aproveite contêineres e algoritmos STL
- Use C++ moderno em vez de padrões C
