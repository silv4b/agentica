# Blazor

## Comandos

- Executar: `dotnet run`
- Testar: `dotnet test`
- Linter: `dotnet format`
- Build: `dotnet build`

## Estilo de Código

- Use componentes Blazor com arquivos `.razor`
- Prefira isolamento CSS para estilos de componente
- Use injeção de dependência em componentes Blazor

## Melhores Práticas

- Escolha o modo de renderização (Server vs WebAssembly) com cuidado
- Use `@rendermode` para renderização interativa no servidor
- Implemente gerenciamento de estado com Fluxor ou serviços customizados
- Use virtualização para listas grandes
- Lidere validação de formulários com DataAnnotations
