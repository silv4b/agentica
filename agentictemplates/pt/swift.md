# Swift

## Comandos

- Executar: `swift run`
- Testar: `swift test`
- Build: `swift build`
- Formatar: `swift format`

## Estilo de Código

- Siga as Swift API Design Guidelines
- Use Swift Package Manager para dependências
- Prefira value types (structs) em vez de reference types (classes)
- Use optionals com tratamento adequado de nil
- Use programação orientada a protocolos

## Melhores Práticas

- Use `Codable` para serialização JSON
- Use `async`/`await` para concorrência
- Prefira SwiftUI em vez de UIKit para novos projetos
- Use atributo `@main` para pontos de entrada
