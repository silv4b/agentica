# Flask

## Comandos

- Executar: `flask run` ou `python app.py`
- Testar: `pytest`
- Linter: `ruff check .`

## Estilo de Código

- Siga PEP 8
- Use blueprints para organização modular
- Prefira o padrão factory para criação da app
- Use type hints para todas as funções públicas

## Melhores Práticas

- Use Flask-SQLAlchemy para acesso a banco de dados
- Use Flask-Migrate para migrações de schema
- Implemente handlers de erro e helpers abort adequados
- Use Flask-Login para autenticação
- Estruture com padrão application factory
