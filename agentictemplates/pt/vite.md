# Vite

## Comandos

- Executar: `npm run dev` ou `npx vite`
- Build: `npm run build` ou `npx vite build`
- Preview: `npm run preview`

## Melhores Práticas

- Use o tratamento de assets e processamento CSS integrado do Vite
- Aproveite HMR para feedback rápido de desenvolvimento
- Use variáveis de ambiente com `import.meta.env`
- Configure alvos de build para compatibilidade de navegadores
- Use plugins do ecossistema oficial Vite
