# C#

## Estilo de Código

- Siga as convenções de código Microsoft C#
- Use `var` quando o tipo for óbvio
- Use membros com corpo de expressão para membros simples
- Prefira tipos `record` para dados imutáveis
- Use tipos de referência anuláveis

## Melhores Práticas

- Use async/await para operações de I/O
- Prefira LINQ em vez de loops imperativos
- Use injeção de dependência para baixo acoplamento
- Implemente `IDisposable` para recursos não gerenciados
- Use `System.Text.Json` em vez de Newtonsoft.Json
