# React

## Comandos

- Executar: `npm run dev`
- Build: `npm run build`
- Testar: `npm run test`
- Linter: `npm run lint`

## Estilo de Código

- Use functional components with hooks
- Use TypeScript for type safety
- Follow the DRY principle
- Use CSS modules or Tailwind for styling
- Keep components small and focused

## Melhores Práticas

- Lift state up only when necessary
- Use `React.memo` and `useMemo` sparingly and intentionally
- Use custom hooks to encapsulate reusable logic
- Avoid prop drilling with context or composition
- Use error boundaries to catch rendering errors
- Prefer controlled components in forms
